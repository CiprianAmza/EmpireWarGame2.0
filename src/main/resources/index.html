EmpireWar is a wonderful game with Strategy, Building, War and Management elements well embedded into the core game-play. This cool game is all about waging epic wars against the enemies, building structures, raising huge armies, advancements in research and development and it offers an amazingly immersive game-play. Start building your village from scratch using Clay, Wood, Stone or Iron

Work for resources. Raise an Empire. Create soldiers. Lead an army.

Win battles - Conquer the world!
